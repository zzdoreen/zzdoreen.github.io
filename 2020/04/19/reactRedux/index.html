react-redux的简单使用

2020-04-19
React
Word count: 512   |   Reading time≈ 2 min

简介

redux为react提供的一个专用库
react-redux组成：

  1. UI组件（presentational component）

    • 只负责ui的呈现，不带有任何业务逻辑
    • 没有状态（不使用this.state这个变量）
    • 所有数据都由参数（this.props）提供
    • 不使用任何Redux的API

  2. 容器组件（container component）

    • 负责管理数据和业务逻辑，不负责UI的呈现
    • 带有内部状态
    • 使用Redux的API

安装

npm i react-redux –save

配合redux使用

npm i redux –save

使用

全局配置

利用<Provider>组件来实现容器组件获取state对象
在src下的index.js全局进行配置

组件配置

connect方法用于从UI组件生成容器组件。
在组件内部引入connect方法

connect方法接收两个参数：

  1. mapStateToProps
  2. mapDispatchToProps
    它们定义了ui组件的业务逻辑
    前者负责输入逻辑 => state映射到ui组件的参数（props）
    后者负责输出逻辑 => 用户对ui组件的操作映射成Action
