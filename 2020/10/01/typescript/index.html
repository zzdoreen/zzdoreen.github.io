typescript基础语法

变量声明

变量名 : 类型 = 值

接口

为类型命名 为代码定义契约
定义参数类型 => interface 接口名 { 属性名: 类型 , … }
定义函数类型 => interface 接口名 { {参数名: 类型,…}: 返回类型 }
可索引类型 => interface 接口名 { [ index: 索引类型 ]: 值类型 }
类类型 => interface 接口名 { 属性名: 类型 }
接口继承 => interface 接口名 extends 父级接口名 { 属性名: 类型 }

函数类型定义

function 函数名( 参数名: 参数类型,… ): 函数返回类型 { return … }
